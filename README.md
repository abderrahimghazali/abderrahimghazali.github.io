# abderrahimghazali.github.io

Personal portfolio site for [Abderrahim GHAZALI](https://abderrahimghazali.github.io) — software engineer specializing in AI agent development, automation, and enterprise Drupal architecture.

Built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com). Deployed via GitHub Pages.

## Stack

- **Framework:** Astro (static output)
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages (GitHub Actions)

## Project Structure

```
/
├── public/          # Static assets (favicon, robots.txt, sitemap, llms.txt)
├── src/
│   ├── components/  # Astro components (TypeWriter, StructuredData)
│   ├── layouts/     # Base layout
│   ├── pages/       # index.astro, about.astro
│   └── styles/      # global.css
└── package.json
```

## Development

```sh
npm install
npm run dev       # localhost:4321
npm run build     # build to ./dist/
npm run preview   # preview production build
```
